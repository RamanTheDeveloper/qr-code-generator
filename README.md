# QR Code Generator
------------

This application was designed to quickly run the application and generate QR codes for Sppedcubing Belgium to be used when generating posters for the schedules and groups.

In this application, the user can paste a link into the input field and a QR code is generated that can used to lead to this website. The QR code is directly downloaded in PNG format. You can name the file before generating the QR code.

It is possible that your computer shows a pop up when running this application due to security reasons, therefore you can run it as administrator or run it normally and click continue when the pop-up shows up. The source code is available to be taken a look at if there is a need (app.py).

There are 2 ways to run the application:
- Go to the online version: https://amber-wreath-p7qr.here.now/
- Run it locally on your computer

If you want to run it locally, then following the below steps: \
1. in Github, click on the Code dropdown
2. Click on 'Download ZIP'
3. Once the zip file is downloaded, unzip the folder
4. In the directory, you should look for a file called 'app.exe', right click on the file and run it as administrator
5. A web browser should open with displaying the application (check url it should be localhost)
